Assignment

Data

In the AidData dataset, each row represents a financial transaction between two countries. The dataset contains the following attributes:

  • Year: year of the commitment
  • Donor: country providing the financial resource
  • Recipient: country or organization receiving the money
  • Commitment Amount: the total amount of financial resources provided
  • Coalesced Purpose Name: the purpose of the transaction

Goal

Your goal is to create 3 independent visualizations of the same data set, each one with the intent of answering the questions stated below. For each numbered visualization, you should be able to create a data visualization that answers all of the questions specified.

The simplified and reduced and full datasets can be found here:

Simplified Data Full Dataset

Visualization 1

Create an overview of the relationships between countries so that it is possible to see who donates to whom and how much. The main question one should be able to answer is:

  • Who are the major donors and to which countries do they donate the most and how much?
  • Conversely, who are the major receivers and which countries do they receive from the most and how much?

We only care about the top 10 recipients and the top 20 donors over time for this question.

In this matrix visualization, the data is ordered by total amount donated by the top 20 donor countries.

Visualization 2

  • Considering only the top 5 purposes of donation, how does the relationship between countries look like in terms of purposes? What composition of purposes do the donations between each pair of countries have?
  • Are there countries that donate to a given country using multiple purposes? Or do counties always donate using one single purpose when donating to another country?

The same as the previous question, we only care about the top 10 recipients and the top 20 donors here.

For each recipient country, there is a pie chart for its donor country's donation purposes.
